# ⏪ Delete messages before

{% hint style="danger" %}
**This option isn't available anymore.**
{% endhint %}

Use this setting to delete only messages sent BEFORE a specific message.

Click the "delete messages before" button and enter the message ID.

Enter `0` to disable.

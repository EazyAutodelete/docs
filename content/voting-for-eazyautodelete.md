## Troubleshooting Voting

### "I voted but didn't get the @Voted role"

**Possible causes:**
- You're not in the Support Server
- Discord/Top.GG synchronization delay
- Role assignment bot is offline

**Solutions:**
1. Make sure you've joined the [Support Server](https://eazyautodelete.xyz/discord/)
2. Wait a few minutes for synchronization
3. Try leaving and rejoining the support server
4. Contact support if issue persists

### "I can't vote again yet"

**Cause:** You must wait 12 hours between votes

**Solution:** 
- Check when you last voted on your Top.GG profile
- Set a reminder for 12 hours after your last vote
- Enable Top.GG vote reminders

### "The vote button isn't working"

**Possible causes:**
- Browser issue
- Top.GG temporary problem
- AdBlocker interference

**Solutions:**
1. Try a different browser
2. Disable adblockers temporarily
3. Clear browser cache and cookies
4. Try again later if Top.GG is having issues
